### Desktop / Laptop
```
1. npm run dev
2. Abre http://localhost:5173
3. Lee [GUIA_RAPIDA.md](GUIA_RAPIDA.md)
4. Haz cambios en VS Code
5. Ver resultados en tiempo real
```

### Necesito hacer cambios desde otro lugar
```
1. Acceso SSH a servidor
2. Git pull
3. Editar con nano/vim
4. npm run build (después de cambios)
```
